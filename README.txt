Deploy Static Website on AWS

PROJECT DESCRIPTION
===================
This project demonstrates how to deploy a static website on Amazon Web Services (AWS). In simple terms, a static website is a collection of web pages that display the same content to all visitors and don't require a database or server-side processing. 

The website is hosted using AWS services including:
- Amazon S3 (Simple Storage Service): A cloud storage service that stores all the website files securely
- CloudFront: A content delivery network (CDN) that ensures your website loads quickly for users around the world
- IAM (Identity and Access Management): Controls who has access to the website resources

This project showcases best practices for cloud deployment, security, and website distribution on a global scale.


PROJECT FILES
=============
The files included are: 

- index.html - The main homepage of the website
- 404.html - Custom error page displayed when a page is not found
- /img/ - Folder containing background images and visual assets
- /vendor/ - Third-party libraries including Bootstrap (CSS framework), Font Awesome (icons), and jQuery (JavaScript library)
- /css/ - Stylesheet files for website styling and layout
